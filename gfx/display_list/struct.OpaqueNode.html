Struct gfx::display_list::OpaqueNode[src]

pub struct OpaqueNode(pub uintptr_t);

An opaque handle to a node. The only safe operation that can be performed on this node is to compare it to another opaque handle or to another node.

Because the script task's GC does not trace layout, node data cannot be safely stored in layout data structures. Also, layout code tends to be faster when the DOM is not being accessed, for locality reasons. Using OpaqueNode enforces this invariant.
