Enum script::dom::bindings::callback::ExceptionHandling[src]

pub enum ExceptionHandling {
    ReportExceptions,
    RethrowContentExceptions,
    RethrowExceptions,
}

The exception handling used for a call.

Variants

ReportExceptions

Report any exception and don't throw it to the caller code.

RethrowContentExceptions

Throw an exception to the caller code if the thrown exception is a binding object for a DOMError from the caller's scope, otherwise report it.

RethrowExceptions

Throw any exception to the caller code.
