A trait to convert JSVals to Rust types.

Required Methods

fn from_jsval(cx: *mut JSContext, val: JSVal, option: T) -> Result<Self, ()>

Convert val to type Self. Optional configuration of type T can be passed as the option argument. If it returns Err(()), a JSAPI exception is pending.
