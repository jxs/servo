Encapsulates a type that can be traced but is boxed in a type we don't control (such as RefCell).

Wrap a field in Traceable and implement the Encodable trait for that new concrete type to achieve magic compiler-derived trace hooks.

We always prefer this, in case the contained type ever changes to something that should be traced. See more details: mozilla#2662.
