Encapsulates a type that cannot easily have Encodable derived automagically, but also does not need to be made known to the SpiderMonkey garbage collector.

Use only with types that are not associated with a JS reflector and do not contain fields of types associated with JS reflectors.

This should really only be used for types that are from other crates, so we can't implement Encodable. See more details: mozilla#2662.
