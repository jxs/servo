Fields

id

Pipeline id associated with this page.

subpage_id

Subpage id associated with this page, if any.

last_reflow_id

Unique id for last reflow request; used for confirming completion reply.

frame

The outermost frame containing the document, window, and page URL.

layout_chan

A handle for communicating messages to the layout task.

layout_join_port

fn reflow(&self, goal: ReflowGoal, script_chan: ScriptChan, compositor: &ScriptListener)

Reflows the page if it's possible to do so. This method will wait until the layout task has completed its current action, join the layout task, and then request a new layout run. It won't wait for the new layout computation to finish.

If there is no window size yet, the page is presumed invisible and no reflow is performed.

This function fails if there is no root frame.
